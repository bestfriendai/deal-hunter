# Assets Required

This folder should contain the following files for the app to build correctly:

## Required Files

1. **icon.png** - App icon (1024x1024 recommended for Expo)
2. **splash.png** - Splash screen image
3. **favicon.png** - Web favicon (48x48 or 512x512)

## Notes

- These files are referenced in `app.json` but are currently missing
- Generate these using: https://expo.dev/@username/project-name (after publishing)
- Or use `npx expo export` to generate web assets

## Color Theme
- Primary color: #FF6B35 (orange)
- Background: #F8F9FA
